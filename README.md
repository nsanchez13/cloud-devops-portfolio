# Cloud & DevOps Portfolio

Este repositorio reúne mis proyectos orientados a Cloud, DevOps e infraestructura. El objetivo es demostrar habilidades prácticas en automatización, despliegue, infraestructura como código y operación de servicios.

Perfil: titulado en ASIR, enfocando mi carrera hacia roles de Cloud / DevOps y construyendo proyectos reales con Terraform, GitHub Actions, Docker y plataformas cloud.

## Tecnologías principales

- Terraform
- GitHub Actions
- Docker
- Linux / Bash
- AWS, GCP y Azure
- Git y control de versiones

## Proyectos

### 1. Terraform Lab GCP
Repositorio: [terraform-lab-gcp](https://github.com/nsanchez13/terraform-lab-gcp)

Proyecto de infraestructura como código para desplegar una base cloud reproducible usando Terraform sobre Google Cloud. Incluye estructura de ficheros, variables, comandos básicos y guía de ejecución.

### 2. GitHub Actions CI/CD
Repositorio: [github-actions-cicd](https://github.com/nsanchez13/github-actions-cicd)

Pipeline de integración continua y automatización para validar cambios, ejecutar linting y preparar flujos básicos de testing y despliegue automático.

### 3. Docker Deployment Lab
Repositorio: [docker-deployment-lab](https://github.com/nsanchez13/docker-deployment-lab)

Proyecto centrado en contenedorización de una aplicación simple, ejecución local y base para despliegue controlado (local o cloud).

## Qué quiero demostrar con este portfolio

- Capacidad para construir proyectos técnicos de principio a fin
- Criterio para documentar decisiones técnicas y trade-offs
- Conocimientos de automatización e infraestructura como código
- Base sólida de sistemas y administración heredada de ASIR
- Evolución hacia un perfil Cloud / DevOps profesional
